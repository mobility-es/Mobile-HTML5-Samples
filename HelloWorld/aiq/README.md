In this directory should be copied the Core API library, plus the framework files inside subfolder "app".
Please copy the conent of aiq folder from AIQ Mobile-HTML5-SDK to this folder. 

Note that the app folder is required only if you are using AIQ HTML5 app framework to develop your application. 

The contents should then look like this:
